How to connect Docker with Pycharm

Step 1. Install Pycharm and Docker

Download Pycharm here and Docker here.

Run Docker and Ensure you have a stable Internet connection.

Step 2. Configure the Docker daemon connection settings

a. In the Settings/Preferences dialog ⌘,, select Build, Execution, Deployment | Docker.

b. Click the "Add button" to add a Docker configuration and specify how to connect to the Docker daemon. The connection settings depend on your Docker version and operating system. For more information, see Docker configuration. The Connection successful message should appear at the bottom of the dialog.

Step 3. Add a configuration for the current project

Click the "Add Configuration" button on the toolbox, then click the "Add" button --> Docker --> Dockerfile. Select the Dockerfile, finally click "OK" button.

Step 4. Connect to the Docker daemon

The configured Docker connection should appear in the Services tool window (View | Tool Windows | Services or ⌘8). Select the Docker node Docker node, and click The Connect button, or select Connect from the context menu.

The Services tool window (View | Tool Windows | Services or ⌘8) enables you to pull and push images, create and run containers, manage Docker Compose, and so on. As with other tool windows, you can start typing the name of an image or container to highlight the matching items.
