Set-up

Thanks to Bootstrap and Github pages, our website can be built and deployed easily.

1. Fork our repository.

Our code is hosted on Github, you can easily fork it to your account.

2. Edit html files.

With Bootstrap, we can create beautiful UI easily. More components can be found here.

3. Public using Github page.
After you push the code to the github repository, just set the Github page in the settings, and you can get a free website space. You can even bind a domain name!
